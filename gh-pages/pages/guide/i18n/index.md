# i18n

By default, the language of milkdown is currently English.
If you want to use other language, you can configure the presets and plugins to support it.

## preset-commonmark

-   image
    -   input.placeholder
    -   input.buttonText
-   link
    -   input.placeholder
    -   input.buttonText

## plugin-math

-   mathInline
    -   placeholder.empty
    -   placeholder.error
    -   input.placeholder
-   mathBlock
    -   placeholder.empty
    -   placeholder.error

## plugin-diagram

-   diagramNode
    -   placeholder.empty
    -   placeholder.error
